## Discuz 新浪微博插件

#### 插件功能亮点：

1. 实现免注册微博一键登录论坛, 也可选择注册论坛绑定微博
2. 用户微博绑定论坛以后发一条微博(微博内容站长设定)
3. 微博勋章、多种积分奖励措施、微博绑定任务， 增加会员活跃度
4. 同步主题、文章、日志、分享、记录和广播到论坛， 并且实现同步格式站长自定义
5. 同步微博内容的评论和转发回流到论坛
6. 分享微博的评论和转发回流到论坛


#### 安装

* DZ版本要求: Discuz X3,X3.1,X3.2  且 PHP支持curl、mbstring库
